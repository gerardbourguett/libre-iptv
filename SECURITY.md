# Política de seguridad

## Versiones soportadas

| Versión | Soporte          |
|---------|------------------|
| 0.2.x   | ✅ Con soporte   |
| < 0.2.0 | ❌ Sin soporte   |

## Reportar una vulnerabilidad

Si descubres una vulnerabilidad de seguridad en IPTV Player, te pedimos que la reportes de manera responsable.

### Proceso

1. **No abras un issue público.** Los problemas de seguridad no deben divulgarse en issues públicos para evitar exponer a los usuarios a riesgos innecesarios.
2. Envía un correo electrónico detallado a la dirección de contacto del proyecto describiendo la vulnerabilidad.
3. Incluye los pasos para reproducirla, el impacto potencial y, si es posible, una sugerencia de mitigación.
4. Espera una confirmación de recepción en un plazo de 72 horas.

### Compromiso

- Revisaremos todos los reportes de seguridad en un plazo máximo de 7 días.
- Mantendremos una comunicación abierta contigo durante el proceso de investigación.
- Una vez solucionada la vulnerabilidad, publicaremos un aviso de seguridad y liberaremos una versión corregida.

Agradecemos tu ayuda para mantener IPTV Player seguro para todos.
